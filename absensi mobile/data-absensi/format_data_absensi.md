# FORMAT DATA ABSENSI EXCEL

## Struktur Kolom yang Dibutuhkan

Pastikan setiap file Excel absensi bulanan memiliki kolom-kolom berikut:

| Kolom | Deskripsi | Contoh |
|-------|-----------|--------|
| Tanggal | Tanggal lengkap | 2025-09-01 |
| Hari | Nama hari | Senin |
| Total Pegawai | Jumlah seluruh pegawai | 142 |
| Hadir | Jumlah yang hadir | 135 |
| Izin | Jumlah yang izin | 3 |
| Sakit | Jumlah yang sakit | 2 |
| Cuti | Jumlah yang cuti | 1 |
| Tanpa Keterangan | Jumlah tanpa keterangan | 1 |
| Dinas Luar | Jumlah dinas luar | 0 |

## Catatan Penting

1. Pastikan format tanggal konsisten (YYYY-MM-DD)
2. Semua kolom harus terisi (gunakan 0 jika tidak ada)
3. Simpan dalam format .xlsx atau .csv
